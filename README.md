关于截波新增电源电缆头改造：
原因分析：在于接地线的电位抬高，导致航插头中火线和地线打火，较大能量导致航插头烧毁。


整改方案：
1.整体去除此接地线，在电缆上标注，必须两头都断电情况方可进行插拔，只连接三根火线，并拆除截波处航插接地环，增加绝缘距离。

2.现有电源隔离回路移动到热继电器后，拆除原端子中三个压敏，“隔离变压器”屏蔽层，零线，铁芯全部悬浮处理。
	

1.电机处串入隔离变压器模块
2.拆除端子中的三个压敏	1.更换此图中压敏电阻为商用防浪涌模块
2.原有N，PE悬空处理

3.更改之后，脉冲放大器供电仍然是使用从控制柜输入的电源，此时，给脉冲供电增加一个单独的380V:220V单相变压器，此变压器需要承受截断瞬间电源和截波底座之间的压差，隔离电压需要做到5kV以上。用于脉放盒供电。为了避免多点接地带来的复杂工况，将AX5盒，脉冲放大器，此新增隔离变压器均放置在绝缘板上。并采用绝缘螺丝进行固定。原AX5航插中，强电部分进行拆除。

增加新隔离变压器，用于脉冲放大器的保护

	
原控制系统AX5电缆走线	更改后的系统AX5电缆走线


关于光纤转换模块8541损坏
采用现在可以正常运行的方案，将8541从控制柜上绝缘固定，采用编织线固定其电源和RS232端口电位，并在串口线上串联串口隔离装置，防止引发PLC模块的损害。同时我们也将提供光纤转换模块备件。


关于整体系统隔离变
在一点接地情况下，地电位的提高按以前项目经验来看可控，控制柜的元器件损害的情况很少，在改进方面，更换原有电源保护压敏电阻，增加控制柜过压保护容量。

替换原有压敏电阻为20kA防浪涌器件
